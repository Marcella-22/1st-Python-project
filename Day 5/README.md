# Power generator

Description : This project is a simple python programme that generates the powers of a number entered by the user in a specific range.It uses a 'while' loop to repeatedly calculate and display the powers until a specified limit.

Language: Python

How to run: open an online python compiler. copy and paste the code. click the run button.Enter a number when prompted.

```python
base = int(input("Enter a number in range 1,5: "))
power = 1
```
